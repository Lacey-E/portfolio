Uploading multiple files Files can be uploaded using an HTML input element, FormData() and fetch().



The Headers interface allows you to create your own headers object via the Headers() constructor. A headers object is a simple multi-map of names to values:

 const content = 'Hello World';  const myHeaders = new Headers();  myHeaders.append('Content-Type', 'text/plain');  myHeaders.append('Content-Length', content.length.toString());  myHeaders.append('X-Custom-Header', 'ProcessThisImmediately');



headers' objects have a guard property. This is not exposed to the Web, but it affects which mutation operations are allowed on the headers object.
