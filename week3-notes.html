Object are used in the real world to represent entities. OOP means Object oriented programming.

To create an object literal, simply enter a pair of curly braces. The following example creates an empty object that is assigned to the variable spiderman:

 const spiderman = {};

You can access the properties of an object using the dot notation

 superman.name
 'Superman'

OR use the brackets

 superman["real" + " " + "name"] // the property is built using string concatenation
  "Clark Kent"

This In Js:

To access the object, a method can use the this keyword. The value of this is the object “before dot”, the one used to call the method.
This when called in the instance of an object, then this becomes the instance

  function Human (age) {
  this.age = age
  }

 let greg = new Human(22)
 let thomas = new Human(24)
 console.log(greg) // this.age = 22
 console.log(thomas) // this.age = 24

Summary:

*Copied from the Notes

  • Functions that are stored in object properties are called “methods”.
  • Methods allow objects to “act” like object.doSomething().
  • Methods can reference the object as this.
  • The value of this is defined at run-time.
  • When a function is declared, it may use this, but that this has no value until the function is called.
  • A function can be copied between objects.
  • When a function is called in the “method” syntax: object.method(), the value of this during the call is object.
  • Please note that arrow functions are special: they have no this. When this is accessed inside an arrow function, it is taken from outside.
